# AI Property Assistant

AI Property Assistant is a property lead management application built with Next.js and deployed on Vercel.

## Features

* Property inquiry form
* Real-time Telegram notifications
* Responsive modern UI
* Serverless API routes
* Vercel deployment

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* Telegram Bot API
* Vercel

## Live Demo

https://v0-ai-property-asisant.vercel.app

## GitHub Repository

https://github.com/dimasn130397/v0-ai-property-asisant

## Workflow

User submits inquiry form
→ Next.js API Route
→ Telegram Bot API
→ Instant notification delivered
